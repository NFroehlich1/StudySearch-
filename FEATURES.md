### 1. **Course Editing with Color Selection**

- Add ECTS credits to each course

- Progress tracking toward 30 ECTS recommendation

- Two pre-configured semesters (WS 2024, SS 2025)

- Full Markdown support in course notes
- Live preview of Markdown content
- Supports bold, italic, links, lists, code blocks
- Store detailed course information and personal notes

### 5. **Enhanced Recommendations Panel**
- Edit button for quick access to course editor
- Direct navigation to PDF handbook pages
- Color-coded course cards

### 6. **Semester Plan Tab**
- New dedicated tab for semester planning
- Visual organization of courses by semester
- ECTS tracking and progress visualization
- Add/remove courses from semesters
